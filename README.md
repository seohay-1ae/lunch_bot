# lunch_bot

## 프로젝트 개요

사이드 프로젝트: 메뉴 추천 디스코드 봇

**lunch_bot**은 디스코드 프로그램에서 구동되는 봇으로,
명령어를 통해 메뉴를 추가, 삭제, 조회 할 수 있고, 랜덤 메뉴 추천을 받을 수 있습니다.
학원에서 매일 점심 메뉴 고민하기가 힘들어 만들게 되었습니다.

### 사용 언어 및 기술 스택

- 백: Python + discord.py
- 데이터: JSON 파일
- 배포: proxmox 구동
